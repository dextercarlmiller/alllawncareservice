# ALL Lawn Care Service — Website
Fort Wayne Web Studio client build.

## Deploy to Cloudflare Pages
1. Push this repo to GitHub
2. Cloudflare Pages → New Project → Connect Git → select repo
3. Build settings: Framework = None, Build command = (blank), Output = /
4. Production branch: main

## After launch — owner setup steps
1. Replace `REPLACE_WITH_FORMSPREE_ID` in index.html with a real Formspree endpoint (free at formspree.io)
2. Replace `G-XXXXXXXXXX` with Google Analytics measurement ID
3. Point fwlawns.com DNS to Cloudflare Pages
4. Verify Google Business Profile links to fwlawns.com
